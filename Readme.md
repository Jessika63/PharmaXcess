### Pre-commit

#### Install

Install everything for pre-commit with this command line:

```bash
bash distributeur/backend/scripts/pre-commit/install_pre-commit.sh
```

#### Try

Try pre-commit with this command line:

```bash
pre-commit run --all-files
```
